# **GET** `/story/{id}`

## Request

### Параметры

* **id** - id выбранной истории (uuid)

### Query параметры:

* **viewer_id** - id пользователся просматривающего историю  (required, uuid)

### Особенности поведения

Каждый раз, когда будет дёргаться ручка, количество `views` увеличивается на 1.
